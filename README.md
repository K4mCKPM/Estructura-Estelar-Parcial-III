# Estructura-Estelar-Parcial-III
Código y datos usados para el desarrollo del tercer parcial de la materia de Estructura Estelar.

Dentro del código de ['Lane_Emden.py'](Lane_Emden.py) se realizó el desarrollo numérico obtenido de [1], desde el cuál se calcularon las soluciones de los índices politrópicos entre 0 y 4, desde los cuales se desarrolló la densidad y la presión al interior de estas distintas polítropas, comparandolas con el *Model S* de Jørgen Christensen-Dalsgaard [2], obteniendo las siguientes gráficas:

![Densidad Fraccionaria](https://github.com/K4mCKPM/Estructura-Estelar-Parcial-III/blob/main/Densidad_Fraccionaria.png)


### Para la Presión fraccionaria
![Presión Fraccionaria](https://github.com/K4mCKPM/Estructura-Estelar-Parcial-III/blob/main/Presion_Fraccionaria.png)

Así mismo, se observó que los mejores acercamientos al *Model S* eran para índices politrópicos entre 3.0 y 4.0, por lo tanto se realizó el mismo desarrollo para la densidad y la presión, pero en este caso con valores que se comportaran similar al *Model S*.
